# Examples of Correct McKinsey-Style Slides

## Example 1: Title Construction

### ❌ WRONG (Generic topic label):
```
Dutch Hydrogen Strategy
```

### ✅ CORRECT (Insight-driven "so-what" title):
```
The Netherlands can unlock €15B in economic value by 2035 through prioritized 
investment in three green hydrogen corridors
```

### ✅ CORRECT (Another example):
```
Accelerating electrolyzer deployment by 2 years would reduce import dependency 
by 40% and create 12,000 new jobs in the northern provinces
```

**Rule**: The title should tell the reader the key takeaway. A reader who only sees the title should understand the main message.

## Example 2: Slide Structure (ASCII representation)

```
┌──────────────────────────────────────────────────────────┐
│  [Insight-driven title in Georgia, 24pt, bold, navy]     │
│  ─────────────────────────────────────────────────────── │  ← thin grey divider (1pt)
│                                                          │
│  [Clean body content area]                               │
│  • Structured table with thin grey gridlines             │
│  • OR clean chart with minimal styling                   │
│  • OR organized text blocks with clear hierarchy         │
│                                                          │
│                                                          │
│                                                          │
│  Source: Company reports; McKinsey analysis               │  ← 8pt, grey, left-aligned
│  ─────────────────────────────────────────────────────── │  ← thin grey footer line
│  CONFIDENTIAL                                        1   │  ← footer: left + right
└──────────────────────────────────────────────────────────┘
```

**Background**: Pure white (#FFFFFF)  
**All text**: Navy or grey tones only  
**Lines**: Thin, light grey, never heavy

## Example 3: Color Usage

### Correct palette application:
- Title text: `#003A70` (navy)
- Body text: `#333333` (dark grey)
- Table headers: `#003A70` navy text on `#F5F5F5` very light grey background
- Table gridlines: `#D9D9D9` (light grey), 0.75pt
- Highlighted data: `#0072CE` (McKinsey blue) — used sparingly
- Footer/source text: `#666666` (medium grey)
- All backgrounds: `#FFFFFF` (white)

### NEVER use:
- `#028090` (teal) — this is off-brand
- `#00A896` (seafoam) — this is off-brand  
- `#02C39A` (mint) — this is off-brand
- Any saturated or vibrant fill colors

## Example 4: Data Table Styling

### ❌ WRONG:
- Thick dark borders around cells
- Alternating bright colored rows
- Solid teal/blue cell fills

### ✅ CORRECT:
- Thin grey gridlines (`#D9D9D9`, 0.75pt)
- Header row: light grey fill (`#F2F2F2`) with navy bold text
- Data cells: white fill, dark grey text
- No outer border or very thin outer border
- Clean, scannable, professional

---

## Example 5: Timeline/Process Flow Styling

### ❌ WRONG:
- Thick solid-filled boxes in teal/cyan
- Heavy connector lines
- Dark background behind timeline

### ✅ CORRECT:
- Thin-bordered boxes (0.75pt, light grey) with NO fill or very light grey fill
- Thin connector lines/arrows (0.75pt, grey)
- Navy text labels, grey supporting text
- All on white background
- Minimal, clean, refined appearance

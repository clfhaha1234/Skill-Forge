# ⚠️ CRITICAL CHECKLIST — Verify Before Finalizing Any Slide

Run through this checklist for EVERY slide. These are the most common failure points.

---

## 1. ✅ Background is dark navy `#002244`
- [ ] Slide background is set to `#002244`
- [ ] Background is NOT white, NOT light grey, NOT any light color
- **If the background is white, the slide automatically fails.**

## 2. ✅ Title uses Georgia (serif) font
- [ ] Title font is **Georgia** (serif), NOT Arial, NOT Calibri
- [ ] Title is 22-26pt, Bold, White (`#FFFFFF`)
- [ ] Title is an insight-driven "so-what" statement, not a topic label
- **The serif title font is a key signature of the reference style.**

## 3. ✅ Divider line present below title
- [ ] Thin horizontal line at ~1.1" from top
- [ ] Color: `#4A90D9`, thickness: 1pt

## 4. ✅ Table is minimalist and open (NOT dense)
- [ ] Table has NO full grid borders (no vertical lines, no lines between every row)
- [ ] Table cells have NO colored background fills (no `#003366`, no `#002B4E`)
- [ ] Only ONE thin horizontal line below the header row
- [ ] Rows have generous spacing (0.4"+ height)
- [ ] Table feels airy and open, not dense and heavy
- **A table with full grid lines and colored cells automatically fails.**

## 5. ✅ No overlapping elements
- [ ] Content ends by 6.3" from top
- [ ] Source note starts at 6.5" from top
- [ ] There is visible space between the last content row and the source text
- [ ] Footer is at 7.0" from top
- **Source text overlapping table content automatically fails.**

## 6. ✅ Professional polish
- [ ] Generous whitespace throughout — nothing feels cramped
- [ ] All text is readable and well-spaced
- [ ] Consistent alignment throughout
- [ ] Footer reads "McKinsey & Company  |  [#]" at bottom-right

---

## Quick Summary of the 3 Biggest Mistakes to Avoid:
1. **Wrong title font** → Must be Georgia (serif), not Arial/Calibri
2. **Heavy/dense table** → Must be minimalist with no grid lines, no cell fills
3. **Overlapping text** → Source must be well below content with visible gap

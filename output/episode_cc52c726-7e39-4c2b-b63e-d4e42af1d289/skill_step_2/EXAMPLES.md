# Examples — Reference Slide Patterns

These examples show the CORRECT structure for slides following the McKinsey reference style.

---

## Example 1: Data Table Slide (MINIMALIST OPEN TABLE STYLE)

```
┌──────────────────────────────────────────────────────────────────┐
│  Background: #002244 (dark navy) — ALWAYS dark, NEVER white      │
│                                                                    │
│  [0.5" margin top: 0.4"]                                          │
│                                                                    │
│  "Global hydrogen production costs have declined 60% since 2015,  │
│   with green hydrogen approaching cost parity by 2030"            │
│   *** Georgia 24pt, Bold, White #FFFFFF ***                       │
│   (MUST be Georgia serif — this is the signature title font)      │
│                                                                    │
│  ─────────────────────────────────────────────────── (thin line)  │
│  (#4A90D9, 1pt, spans margin-to-margin)                           │
│                                                                    │
│  [0.25" gap — breathing room]                                     │
│                                                                    │
│   Technology      2015       2020       2025E      2030E          │
│   (bold, white)   (white)    (white)    (white)    (white)        │
│  ─────────────────────────────────────── (#4A90D9, 0.75pt)       │
│                                                                    │
│   Green H2        $6.0/kg    $3.5/kg    $2.1/kg    $1.4/kg       │
│                                                                    │
│   Blue H2         $2.5/kg    $2.0/kg    $1.6/kg    $1.3/kg       │
│                                                                    │
│   Grey H2         $1.5/kg    $1.2/kg    $1.4/kg    $1.5/kg       │
│                                                                    │
│                                                                    │
│  [generous gap — at least 0.3" between content bottom and source] │
│                                                                    │
│  Source: IEA, BloombergNEF (2024)                                 │
│  (Calibri 8pt, italic, #8899AA, left-aligned, at 6.5" from top)  │
│                                                                    │
│                                    McKinsey & Company    |    1   │
│                                    (Calibri 9pt, #8899AA, right)  │
└──────────────────────────────────────────────────────────────────┘
```

### Key points demonstrated:
- **Dark navy background (#002244) — NOT white**
- **Title uses Georgia (serif) font** — the signature style element
- Insight-driven title (a finding, not a label)
- Thin blue divider line below title
- **MINIMALIST open table — NO grid lines, NO cell backgrounds, NO row striping**
- **Only a single thin horizontal line below the header row**
- **Table cells have NO fill — transparent, showing the navy background**
- **Generous spacing** between rows — table feels airy and open
- **Large gap** between table bottom and source note — no overlapping
- Source note well below content, small and muted
- Footer: "McKinsey & Company | [Page #]" right-aligned

## Example 2: Multi-Column Content Slide

```
┌──────────────────────────────────────────────────────────────────┐
│  Background: #002244 (dark navy) — ALWAYS dark, NEVER white      │
│                                                                    │
│  "Three policy mechanisms account for 85% of global hydrogen      │
│   investment commitments"                                          │
│   *** Georgia 24pt, Bold, White ***                               │
│                                                                    │
│  ─────────────────────────────────────────────────── (#4A90D9)   │
│                                                                    │
│  ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐  │
│  │  Production       │ │  Infrastructure  │ │  Demand-side     │  │
│  │  Tax Credits      │ │  Subsidies       │ │  Mandates        │  │
│  │                   │ │                  │ │                  │  │
│  │  $145B committed  │ │  $85B committed  │ │  $50B committed  │  │
│  │                   │ │                  │ │                  │  │
│  │  • US IRA §45V    │ │  • EU hydrogen   │ │  • EU refinery   │  │
│  │  • EU green deal  │ │    backbone      │ │    mandate       │  │
│  │  • Japan GX       │ │  • US DOE hubs   │ │  • UK transport  │  │
│  │                   │ │                  │ │    targets       │  │
│  └──────────────────┘ └──────────────────┘ └──────────────────┘  │
│  (Boxes: subtle border #4A90D9, no fill or very subtle #002B4E)  │
│                                                                    │
│  [generous gap here — content ends well above source]             │
│                                                                    │
│  Source: Government policy announcements as of Q1 2024            │
│                                    McKinsey & Company    |    2   │
└──────────────────────────────────────────────────────────────────┘
```

### Key points demonstrated:
- **Title uses Georgia (serif) font — always**
- Three-column layout with even spacing
- No decorative clutter — no sidebars or vertical labels
- Content starts well below the title + divider
- Content ends well above the source note — no overlap
- Consistent, restrained color use

## Example 3: How the Minimalist Table Differs from a Dense Table

### ❌ WRONG — Dense, heavy table (DO NOT DO THIS):
```
  ┌─────────────┬──────────┬──────────┬──────────┐
  │ Category    │  Col 1   │  Col 2   │  Col 3   │  ← colored bg #003366
  ├─────────────┼──────────┼──────────┼──────────┤  ← grid lines everywhere
  │ Row 1       │  data    │  data    │  data    │  ← colored bg #002B4E
  ├─────────────┼──────────┼──────────┼──────────┤
  │ Row 2       │  data    │  data    │  data    │  ← colored bg #002244
  ├─────────────┼──────────┼──────────┼──────────┤
  │ Row 3       │  data    │  data    │  data    │  ← colored bg #002B4E
  └─────────────┴──────────┴──────────┴──────────┘
```
Problems: Full grid borders, colored cell fills, cramped rows, heavy appearance.

### ✅ CORRECT — Minimalist open table (DO THIS):
```
  Category        Col 1       Col 2       Col 3
  ─────────────────────────────────────────────────  ← single thin line
  Row 1           data        data        data

  Row 2           data        data        data

  Row 3           data        data        data


                                                     ← generous bottom space
  Source: ...
```
Features: No cell borders, no cell fills, just one thin line below header, generous row spacing, airy/open feel. The navy slide background shows through transparently.

## Common Mistakes to Avoid

| Mistake | Correct Approach |
|---------|-----------------|
| Title in Arial/Calibri (sans-serif) | **Title in Georgia (serif)** — the signature font |
| Title: "Energy Strategy" | Title: "Renewable energy investment must triple by 2030 to meet net-zero targets" |
| White slide background | **Dark navy background (#002244) — ALWAYS** |
| Dense table with full grid lines | **Minimalist table with only a thin header line** |
| Colored cell backgrounds in table | **No cell fills — transparent background** |
| Row striping (#002B4E / #002244) | **No row striping — clean open rows** |
| Source text overlapping table | **Source at 6.5" from top, content ends by 6.3"** |
| Content crammed to bottom edge | **Generous bottom margin — content ends by 6.3"** |
| No divider line | Always add thin #4A90D9 line below title |
| Footer: "Source: XYZ \| CompanyName" | Footer: "McKinsey & Company  \|  1" (sources go separately above) |
| Graphical timeline with shapes | Clean minimalist table with year columns |

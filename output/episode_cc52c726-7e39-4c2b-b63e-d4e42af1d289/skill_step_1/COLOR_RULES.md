# Color Rules — Strict McKinsey Navy Palette

## The ONLY Approved Colors

This is the complete, exhaustive list of colors allowed on any slide.
Do NOT use any color not listed here.

### Backgrounds
| Use | Hex | RGB |
|-----|-----|-----|
| Slide background | `#002244` | (0, 34, 68) |
| Table header bg | `#003366` | (0, 51, 102) |
| Table alt row bg | `#002B4E` | (0, 43, 78) |
| Content box fill (optional) | `#002B4E` | (0, 43, 78) |

### Text Colors
| Use | Hex | RGB |
|-----|-----|-----|
| Title text | `#FFFFFF` | (255, 255, 255) |
| Primary body text | `#FFFFFF` | (255, 255, 255) |
| Secondary text / labels | `#B0C4DE` | (176, 196, 222) |
| Muted text (footer, source) | `#8899AA` | (136, 153, 170) |

### Accent Colors
| Use | Hex | RGB |
|-----|-----|-----|
| Divider line | `#4A90D9` | (74, 144, 217) |
| Highlight / emphasis | `#4A90D9` | (74, 144, 217) |
| Chart primary | `#4A90D9` | (74, 144, 217) |
| Chart secondary | `#6BAAEE` | (107, 170, 238) |
| Chart tertiary | `#B0C4DE` | (176, 196, 222) |

## BANNED Colors — Never Use These

| Banned Color | Hex | Why |
|-------------|-----|-----|
| Neon green | `#02C39A`, `#00FF00`, etc. | Off-brand, too saturated |
| Teal | `#028090`, `#008080` | Not in the navy palette |
| Seafoam | `#00A896` | Not in the navy palette |
| Cyan | `#00FFFF`, `#00CED1` | Too bright, off-brand |
| Orange | `#FF8C00`, `#FFA500` | Not in the navy palette |
| Red | `#FF0000`, `#CC0000` | Too aggressive |
| Bright yellow | `#FFFF00` | Clashes with dark navy |
| White (as background) | `#FFFFFF` background | Background must be dark navy |
| Light grey (as background) | `#F0F0F0`, `#E0E0E0` bg | Background must be dark navy |

## Color Philosophy

- The palette is **monochromatic navy-blue** with white text
- Think: dark suit, crisp white shirt — classic, authoritative, restrained
- When in doubt, use `#FFFFFF` for text and `#4A90D9` for any accent
- Maximum of 3 distinct hues on any single slide (navy, blue, white)

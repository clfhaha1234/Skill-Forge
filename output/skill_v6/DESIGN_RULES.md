# Design Rules — McKinsey Consulting Style (v6)

## Changes from v5
- Add light blue column fill (#E8F0FE) for Blue H2 cells (entire column)
- Add light green column fill (#E6F4EA) for Green H2 cells (entire column)
- Grey H2 column stays white
- CO2 emission numbers displayed in rounded rectangle boxes
- "Low-carbon H2" label with bracket spanning Blue+Green columns
- Fix footnotes: darker grey (#555555), move up slightly
- Title: keep 16pt Georgia Bold, ensure 2 clean lines
- Add CO2 emissions legend in top-right

## Background & Layout
- White background (FFFFFF)
- 16:9 (10" x 5.625")
- Margins: 0.5" left/right

## Color Palette
- Text: black (000000) or dark grey (333333)
- Navy accent: 1B3A5C (header top border, 2pt)
- Blue H2 column fill: E8F0FE (all cells in Blue H2 column)
- Green H2 column fill: E6F4EA (all cells in Green H2 column)
- Grey H2 column: white FFFFFF
- Row label column: white FFFFFF
- Table horizontal borders: D0D0D0 (0.5pt)
- NO vertical borders
- Footnotes: 555555
- Footer: 666666

## Typography
- Title: Georgia Bold, 16pt, black
- Table header: Calibri Bold, 10pt
- Table body: Calibri, 8.5pt, #333333
- Row labels: Calibri Bold, 8.5pt, black
- Footnotes: Calibri, 6.5pt, #555555
- Footer: Calibri, 8pt, #666666

## Table Structure
- Horizontal-only borders (per-cell border arrays)
- 2pt navy top border on header row (as separate LINE shape)
- Header row bottom: 1pt #333333
- Data row borders: 0.5pt #D0D0D0
- Blue H2 cells: fill E8F0FE
- Green H2 cells: fill E6F4EA
- Grey/label cells: fill FFFFFF

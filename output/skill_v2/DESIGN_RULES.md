# Design Rules — McKinsey Consulting Style (v2)

## Changes from v1
- Fix "Low-carbon H2" overlap: move above table as a centered label, not overlapping divider
- Use McKinsey navy (#1B3A5C) for table header row border instead of generic pastels
- Add 2pt navy top border on header row for structural emphasis
- Fix footnote formatting: inline numbers, no awkward line breaks
- Use specific McKinsey blue (#D6EAF8 → #DCE9F5) and green (#D5F5E3 → #E8F6EF) tints
- Add stronger table row borders for structure
- Remove pastel column fills from data cells — use white with subtle alternating rows

## Background & Layout
- White background (FFFFFF)
- 16:9 (10" x 5.625")
- Margins: 0.5" left/right

## Color Palette
- Text: black (000000)
- Navy accent: 1B3A5C (for header borders and emphasis)
- Blue H2 column header: DCE9F5
- Green H2 column header: E8F6EF
- Data cells: white (FFFFFF) — no colored cell backgrounds
- Grey for footer/footnotes: 666666
- Table borders: C0C0C0 (medium grey, 0.75pt)

## Typography
- Title: Georgia Bold, 22pt, black
- Subtitle descriptor: Calibri, 11pt, #666666
- Table header: Calibri Bold, 11pt
- Table body: Calibri, 10pt
- Footnotes: Calibri, 7pt, #666666
- Footer: Calibri, 8pt, #666666

## Title
- Insight "so-what" title, 2 lines max
- Section numbering like "1.1/" prefix matching reference style

## Table Structure
- Header row: 2pt navy (1B3A5C) top border, bold centered text
- Blue H2 header cell: light blue fill (DCE9F5)
- Green H2 header cell: light green fill (E8F6EF)
- All data cells: white background, no colored fills
- Row borders: 0.75pt medium grey (C0C0C0)
- Bold row labels on left column

## Structural Elements
- Thin divider: 1pt dark grey (333333), below title at y=1.15
- "Low-carbon H2" label centered above the 3 H2 columns, NOT overlapping divider
- Footer line: 0.5pt dark grey at y=5.3
- "McKinsey & Company    4" right-aligned at y=5.35
- Footnotes: compact, inline numbering, y=4.95

## Layout
- Title: x=0.5, y=0.2
- Divider: y=1.15
- Column label "Low-carbon H2": y=1.3, centered above columns 2-4
- Table: y=1.55 to ~4.85
- Footnotes: y=4.9
- Footer line: y=5.3
- Footer text: y=5.35

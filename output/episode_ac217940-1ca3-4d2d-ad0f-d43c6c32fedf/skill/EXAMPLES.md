# Examples — McKinsey-Style Slide Patterns

## Example 1: Data-Driven Strategy Slide

### Title (Insight-Driven)
"Green hydrogen production costs must fall below €2/kg by 2030 to achieve cost parity — requiring €15B+ in Dutch infrastructure investment"

### Structure
```
┌─────────────────────────────────────────────────────────────────┐
│  [TITLE - Georgia 26pt bold, navy #192A3E, left-aligned]       │
│  Green hydrogen production costs must fall below €2/kg by 2030  │
│  to achieve cost parity — requiring €15B+ in Dutch investment   │
│  ─────────────────────────────────────────────────────────────  │ ← grey divider line
│                                                                 │
│  [Subtitle - Calibri 14pt bold]                                 │
│  Key investment areas and projected cost trajectory              │
│                                                                 │
│  ┌──────────────────┬──────────┬──────────┬──────────────────┐  │
│  │ Investment Area   │ 2024 (€B)│ 2030 (€B)│ Cost Impact      │  │ ← navy header row
│  ├──────────────────┼──────────┼──────────┼──────────────────┤  │
│  │ Electrolyzer cap. │ 1.2      │ 5.8      │ -45% prod. cost  │  │ ← white row
│  │ Pipeline infra.   │ 0.8      │ 3.2      │ -20% dist. cost  │  │ ← light blue-grey row
│  │ Storage facilities│ 0.3      │ 2.1      │ -15% storage cost│  │ ← white row
│  │ Port terminals    │ 0.5      │ 1.9      │ -10% import cost │  │ ← light blue-grey row
│  └──────────────────┴──────────┴──────────┴──────────────────┘  │
│                                                                 │
│  [Callout box - light blue-grey background]                     │
│  "Total investment gap: €8.5B between current commitments       │
│   and required 2030 targets"                                    │
│                                                                 │
│  Source: IEA World Energy Outlook, 2024; Dutch Ministry of      │
│  Economic Affairs                                               │
│  ─────────────────────────────────────────────────────────────  │ ← footer divider
│  McKinsey & Company                                          1  │ ← footer
└─────────────────────────────────────────────────────────────────┘
```

### Color Usage in This Example
- Background: WHITE (`FFFFFF`)
- Title text: Dark Navy (`192A3E`)
- Divider lines: Grey (`8C9DAD`)
- Table header: Dark Navy background (`192A3E`), white text (`FFFFFF`)
- Alternating rows: White / Light Blue-Grey (`D6E4F0`)
- Body text: Near-black (`333333`)
- Source & footer: Medium Grey (`5A6A7A`)

## Example 2: Comparison / Benchmarking Slide

### Title (Insight-Driven)
"European hydrogen leaders outperform peers by 3x on electrolyzer deployment, with the Netherlands ranking 2nd behind Germany"

### Structure
- White background
- Insight-driven title (Georgia, 26pt, bold, navy)
- Grey horizontal divider line under title
- Structured comparison table with 5–6 columns
- Navy header row, alternating white/light-blue-grey body rows
- Bold or highlighted cells for the key data point (Netherlands row)
- Source line in 8pt grey above footer
- Footer: "McKinsey & Company" left, page number right

## Anti-Patterns — What NOT to Do

### ❌ BAD: Topic-Label Title
"Dutch Hydrogen Strategy"
→ This tells the reader nothing. What ABOUT the strategy?

### ✅ GOOD: Insight-Driven Title  
"The Netherlands' hydrogen strategy faces a €8.5B funding gap that threatens its 2030 climate commitments"
→ This tells the reader the key conclusion.

### ❌ BAD: Dark Background with Neon Colors
Dark teal background with bright mint accents and yellow text.
→ Looks like a tech startup pitch, not a consulting deliverable.

### ✅ GOOD: White Background with Navy/Grey
Clean white background, navy text, grey dividers, light blue-grey table fills.
→ Professional, restrained, authoritative.

### ❌ BAD: Timeline/Infographic Layout
Icons connected by arrows in a horizontal flow.
→ Appropriate for a blog post, not an analytical slide.

### ✅ GOOD: Structured Data Table
Rows and columns with clear headers, aligned numbers, alternating fills.
→ Enables the reader to compare data points rigorously.

### ❌ BAD: No Footer
Slide ends abruptly at the bottom with no attribution or page number.

### ✅ GOOD: Standard Footer
Thin grey line, then "McKinsey & Company" on the left, page number on the right.

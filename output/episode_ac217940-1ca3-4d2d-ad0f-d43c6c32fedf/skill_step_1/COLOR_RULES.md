# Color Rules — McKinsey Standard Palette

## MANDATORY: White Background
The slide background MUST be `FFFFFF` (pure white). No exceptions.

## Primary Palette (use these for 95%+ of the slide)

| Element | Hex Code | RGB | Usage |
|---------|----------|-----|-------|
| Dark Navy | `192A3E` | 25, 42, 62 | Titles, headers, table header bg |
| Black | `000000` | 0, 0, 0 | Alternative for body text |
| Dark Charcoal | `333333` | 51, 51, 51 | Body text |
| Medium Grey | `5A6A7A` | 90, 106, 122 | Subtitles, footnotes, sources, footer |
| Light Grey | `8C9DAD` | 140, 157, 173 | Divider lines, table borders |
| Light Blue-Grey | `D6E4F0` | 214, 228, 240 | Table alternating rows, callout bg |
| McKinsey Blue | `004B87` | 0, 75, 135 | Accent highlights, key data |
| White | `FFFFFF` | 255, 255, 255 | Background, table header text |

## Accent Colors (use sparingly, only when needed for data differentiation)

| Element | Hex Code | Usage |
|---------|----------|-------|
| Dark Blue | `003A6B` | Chart bars, emphasis |
| Steel Blue | `4A7FB5` | Secondary chart element |
| Light Steel | `A8C4E0` | Tertiary chart element |
| Muted Red | `C0392B` | Negative / risk callout only |
| Muted Green | `27AE60` | Positive / growth callout only |

## FORBIDDEN Colors
Do NOT use any of the following:
- `028090` (teal) — too vibrant
- `00A896` (seafoam) — too vibrant  
- `02C39A` (mint) — too vibrant
- Any neon colors
- Any bright/saturated yellows, oranges, or pinks
- Any color not listed in the palettes above

## Contrast Rules
- Text on white background: use `192A3E`, `333333`, or `000000` (all pass WCAG AA)
- Text on navy background (`192A3E`): use `FFFFFF` only
- Source/footnote text (`5A6A7A`) on white background: passes WCAG AA for small text
- NEVER place light-colored text on a dark background outside of table headers
- NEVER place dark text on a dark background

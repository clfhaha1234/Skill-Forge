# Design Rules — McKinsey Consulting Style (v4)

## Changes from v3
- Title font reduced to 16pt (Georgia Bold renders ~2x larger in LibreOffice than expected)
- Remove ALL colored header fills — use white background throughout table
- Only accent: 2pt navy top border on header row (per ref-04 style)
- Highlighted columns use very subtle light blue tint ONLY in the header cell if needed
- Title must fit in 2 lines maximum at the reduced size
- Fix divider/subtitle overlapping with long titles

## Background & Layout
- White background (FFFFFF)
- 16:9 (10" x 5.625")
- Margins: 0.5" left/right

## Layout Positions (STRICT — do not violate)
- Title: x=0.5, y=0.2, h=0.65 (must fit in this box)
- Divider line: y=0.95
- Subtitle: y=1.0, h=0.2
- Navy header border: y=1.25
- Table: y=1.26, h=3.2 max
- Source/footnotes: y=4.55, h=0.3
- Footer line: y=5.1
- Footer text: y=5.15

## Color Palette (RESTRAINED)
- Text: black (000000) or dark grey (333333)
- Navy accent border: 1B3A5C (2pt, header row top only)
- ALL table cells: white (FFFFFF) — NO colored fills
- Table borders: D0D0D0 (0.5pt light grey)
- Footer/footnotes: 666666
- NO red, pink, pastel blue, pastel green fills

## Typography
- Title: Georgia Bold, 16pt, black
- Subtitle: Calibri, 9pt, #666666
- Table header: Calibri Bold, 10pt, black
- Table body: Calibri, 9pt, #333333
- Footnotes: Calibri, 6pt, #666666
- Footer: Calibri, 8pt, #666666

## Title
- Insight "so-what" conclusion
- Section numbering "1.1/" prefix
- Max 2 lines at 16pt

## Table
- NO colored fills anywhere
- 2pt navy (1B3A5C) top border on header row
- Header: bold, centered
- Data: left-aligned, 9pt
- Row borders: 0.5pt light grey (D0D0D0)
- Bold row labels in left column

## Structural
- Divider: 1pt dark grey (333333), full width
- Footer line: 0.5pt dark grey
- "McKinsey & Company    N" right-aligned
- Source line in 6pt grey

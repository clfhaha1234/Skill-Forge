# Design Rules — McKinsey Consulting Style (v7 — Final)

## Background & Layout
- White background (FFFFFF)
- 16:9 (10" x 5.625")
- Margins: 0.4" left/right

## Color Palette (RESTRAINED)
- Text: black (000000) or dark grey (333333)
- Navy accent: 1B3A5C (2pt header top border)
- Column tints: subtle pastel per category (e.g., Blue H2 = DDEAF6, Green H2 = D5EDDB)
- Row labels and Grey columns: white FFFFFF
- Horizontal borders: D0D0D0 (0.5pt) — NO vertical borders
- Footnotes: 555555
- Footer: 666666

## Typography
- Title: Georgia Bold, 16pt, black (max 2 lines)
- Subtitle: Calibri, 8.5pt, #666666
- Table header: Calibri Bold, 9pt
- Table body: Calibri, 8pt, #333333
- Row labels: Calibri Bold, 8pt, black
- Footnotes: Calibri, 6.5pt, #555555
- Footer: Calibri, 8pt, #666666

## Title Style
- Insight "so-what" conclusion (NOT topic label)
- Section numbering "1.1/" prefix
- Max 2 lines at 16pt Georgia Bold

## Table Structure
- Horizontal-only borders (per-cell border arrays)
- 2pt navy (#1B3A5C) top border on header row (separate LINE shape)
- Header row bottom: 1pt #333333
- Data row borders: 0.5pt #D0D0D0
- Left/right borders: invisible (pt: 0)
- Column tints for highlighted categories
- Bold row labels in left column
- Compact row heights — keep text concise to avoid overflow

## Layout Positions
- Title: x=0.4, y=0.25, h=0.65
- Divider: y=0.98 (0.75pt dark grey)
- Label/subtitle: y=1.04
- Navy header border: y=1.25
- Table: y=1.26, h=2.8 max
- Footnotes: y=4.12
- Footer line: y=5.05
- Footer text: y=5.1

## Structural Elements
- CO2/metric legend circle in top-right (optional)
- "Low-carbon H2" or equivalent label centered above relevant columns
- Numbered footnotes in compact 6.5pt
- Footer line: 0.5pt dark grey
- "McKinsey & Company    N" right-aligned

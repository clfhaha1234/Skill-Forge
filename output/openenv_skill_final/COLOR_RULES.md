# Color Rules — Restrained Professional Palette

## Philosophy
Consulting slides communicate authority through restraint. Use the minimum number of 
colors necessary. White space is a feature, not a waste. Every color choice should 
serve a purpose (hierarchy, grouping, or emphasis).

## Approved Color Palette

### Primary Colors (use on every slide)
| Role | Hex | RGB | Usage |
|------|-----|-----|-------|
| Background | `FFFFFF` | 255,255,255 | Slide background — ALWAYS white |
| Primary text | `333333` | 51,51,51 | Titles, body text, labels |
| Secondary text | `666666` | 102,102,102 | Subtitles, secondary labels |
| Muted text | `999999` | 153,153,153 | Footnotes, sources, page numbers |

### Accent Colors (use sparingly)
| Role | Hex | RGB | Usage |
|------|-----|-----|-------|
| Navy | `003A70` | 0,58,112 | Table headers, divider lines, key emphasis |
| Steel blue | `4472C4` | 68,114,196 | Chart bars, secondary accent |
| Light gray | `D6DCE4` | 214,220,228 | Table borders, separator lines |
| Row shading | `F2F2F2` | 242,242,242 | Alternating table rows |
| Light blue tint | `F2F6FA` | 242,246,250 | Callout box backgrounds |

### Colors to NEVER Use
- `028090` (teal) — too bright and informal
- `00A896` (seafoam) — clashes with professional tone
- `02C39A` (mint/green) — too playful
- Any saturated red, orange, or green (unless specifically for RAG status indicators)
- Black `000000` for backgrounds — always use white
- Any neon or high-saturation color

## Color Application Rules

### Maximum 3 non-neutral colors per slide
- Neutrals (white, grays, black) don't count toward the limit
- Navy counts as 1
- Steel blue counts as 1  
- If you need a third, use a muted gold `B8860B` or muted red `C0392B` for emphasis only

### Table Coloring
- Header row: Navy `003A70` fill, white `FFFFFF` text
- Body rows: Alternating `FFFFFF` and `F2F2F2`
- Borders: `D6DCE4` at 0.5–1pt
- Never use colored fills in body cells (except very light tint for emphasis)

### Chart Coloring
- Primary series: `003A70` (navy)
- Secondary series: `4472C4` (steel blue)
- Tertiary series: `A6A6A6` (gray)
- Avoid more than 3–4 series; group or summarize if needed

### Text Emphasis
- Bold text for emphasis (preferred over color changes)
- If color emphasis needed, use navy `003A70` — never a bright color
- Underline sparingly; bold is preferred
